Code Example: Multiply Function

This example demonstrates a function to multiply two numbers. The original code contains a bug where the return statement is missing.

Incorrect Code:


            function multiply(a, b) {
                a*b;
            }

Issue: The function doesn't return any value, so it won't work as expected. We need to add a return statement.

Corrected Code:


            function multiply(a, b) {
                return a * b;
            }

Sample Tests(using Chai Assertion Library):


                    const assert = require("chai").assert;
                    describe("Multiply", () => {
                        it("fixed tests", () => {
                            assert.strictEqual(multiply(1, 1), 1);
                            assert.strictEqual(multiply(2, 1), 2);
                            assert.strictEqual(multiply(2, 2), 4);
                            assert.strictEqual(multiply(3, 5), 15);
                            assert.strictEqual(multiply(5, 0), 0);
                            assert.strictEqual(multiply(0, 5), 0);
                            assert.strictEqual(multiply(0, 0), 0);
                        });
                    });

This code will now correctly multiply two numbers and pass the test cases.
